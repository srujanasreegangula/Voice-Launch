# Voice-Launch
# 🎙️ Voice Launch – Voice-Controlled Application Launcher

Voice Launch is a modern web application that allows users to open websites using simple voice commands. Built with **HTML**, **CSS**, and **JavaScript**, the project uses the **Web Speech API** to recognize speech and perform actions instantly, creating a hands-free browsing experience. Inspired by voice-driven navigation concepts and web automation workflows. ([archive.fossunited.org][1])

Try speaking commands such as:

```text

Open GitHub
Open Gmail
```

The application automatically detects the command and opens the corresponding website.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/srujanasreegangula/Voice-Launch.git
```

### Navigate to the Project Folder

```bash
cd Voice-Launch
```

### Run Locally

Because microphone access requires a secure context, run the project using a local server:

```bash
python -m http.server 5500
```

Then open:

```text
http://localhost:5500
```

in Google Chrome.

You can also use **VS Code Live Server**.

---

## 🔐 Important Note

Voice recognition does **not work** when opening the HTML file directly using:

```text
file://
```

Always run the project using:

```text
http://localhost
```

or

```text
https://
```

to enable microphone access.
